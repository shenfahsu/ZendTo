ZendTo Downloads 

The latest version is 3.54 which I released on 9 July 2010.

The features I have added to the original Dropbox package are:
  1. Active Directory (AD) authentication against multiple forests, so that if your organisation has more than 1 set of usernames in use, your users can use any of the username/password combinations they can remember. Here we have a departmental setup and a University-wide setup, for historical reasons.
  2. Sender email address is checked for people sending files from other organisations. ZendTo handles the problem of receving files from non-local users very well, and it now checks that the sender owns the email address they have supplied.
  3. Sender can write a note to the recipients explaining the files.
  4. Notification emails about pickups correctly report who picked up the files.
  5. Check of sender email address can only be used by real people.
  6. Claim ID and Passcode are only given to authorised users.
  7. Added support for MySQL database as well as SQLite.
  8. Added VMWare virtual machine distributions for easy deployment.
  9. Totally separated user interface from program code for easy customisation.
  10. Progress indicator for users uploading files.
